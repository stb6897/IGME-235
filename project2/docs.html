Documentation

Initial Idea Phase:

Initially, I wanted to have a single image of a dog with a random button. The user could then see what kind of dog came up, and if they liked the breed of dog they could search for more of that kind. I've essentially done that however what's different from the initial idea is there was a second page. Mine only has one page but instead, it dynamically changes the elements depending on what mode it's in.

Prototype Phase:

In this phase, there was just a random button and an image that would update when you pressed the button. There was also a preliminary dropdown box where you could pick a breed and search. The theme was also green at the time. There was quite a bit of whitespace so I tried to get rid of most of it in the final project.

Final Phase:

In the final phase, I put a lot more effort into the style of the website. I changed up the color scheme with something a bit more fun and added animations to the buttons. This is also when I implemented the multiple-image display. At first, I wanted to add a carousel but I decided to dynamically move around the content instead of new pages. For the images themselves, you may have noticed that they are all a uniform size.
Mozilla docs
